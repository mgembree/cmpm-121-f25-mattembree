# Frog Clicker (CMPM-121 Assignment)

A small browser-based clicker game implemented in TypeScript/Deno as a class assignment for CMPM-121. Click the big frog to collect resources and spend them on upgrades. Recent updates added a right-hand inventory sidebar that shows which items you own (emoji + title) and their counts.

## Recent changes (Changelog)

- 2025-11-06:
  Added an Inventory sidebar that displays owned items (emoji + title) along with counts. Also reorganized some sections and moved the `Item` interface to the top for clarity. Inspired by https://riyer4.github.io/cmpm-121-f25-Ria-Iyer-D1/'s demo
  Added Background ambience with swamp noises, Inspired by https://benho612.github.io/CMPM1212-D1-Assignement/ 's demo, including sound effects.
